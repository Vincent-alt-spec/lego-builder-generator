Turn any LEGO set into a custom alternate build using AI.
Live demo: https://lego-builder-generator.streamlit.app/
## What it does
- Fetches real parts from Rebrickable  
- Selects usable parts based on build size  
- Generates step-by-step build instructions  
- Provides build guidance (difficulty, time, stability)

## How to use
1. Enter a LEGO set number  
2. Choose build size (small / medium / large)  
3. Choose a build type (vehicle, robot, structure, etc.)  
4. Click **Generate Build**

## Notes
This tool provides conceptual instructions.  
Some connections may require creative adjustment.
